# Loan Interest
# Problem Statement
This program calculates loan interest by taking the loan amount, interest rate, and loan term from the user. It helps users determine the interest using either simple or compound interest.
# Features
-Accepts loan amount, interest rate, and loan term
-Calculates simple and compound interest
-Displays yearly loan balances
-Allows multiple calculations in one run
-Uses object-oriented programming
# How to run file
1. compile the files(Loan.java & LoanApp.java)
2. run LoanApp.java
